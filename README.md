# Etch-A-Sketch

A browser-based drawing application that creates colorful pixel art as you move your mouse across a grid of squares.

## Features

- **Interactive Grid**: Hover over squares to leave a colored trail
- **Customizable Grid Size**: Set grid dimensions from 1x1 to 100x100
- **Color Modes**: 
  - Blue mode (default) - all squares turn blue
  - Random mode - each square becomes a random RGB color
- **Dynamic Sizing**: Squares automatically resize to fit the container regardless of grid size

## Technologies Used

- HTML
- CSS (Flexbox)
- JavaScript (DOM manipulation, event listeners)
